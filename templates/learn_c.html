C Tutorial

1. C Overview

What is C?

C is a powerful, general-purpose programming language developed by Dennis M. Ritchie at Bell Labs in 1972. It is a procedural language known for its efficiency, low-level memory manipulation, and portability, making it a foundational language for many modern systems and software.

Why Use C?

C was initially created for system development and produces code that runs nearly as fast as assembly language. It is widely used for:

  • Operating Systems (UNIX, Linux, Windows kernels)
  • Embedded Systems and Device Drivers
  • Compilers, Databases, and Language Interpreters
  • High-performance computing

2. Basic Syntax

"Hello, World!" Program

The entry point for every C program is the `main()` function. The `#include ` is a preprocessor directive that includes the standard input-output library, providing access to functions like `printf()`.


#include 

int main() {
   /* my first program in C */
   printf("Hello, World! \n");
   
   return 0;
}

3. Data Types

Fundamental Data Types

C has a set of basic data types that are used to declare variables. The type of a variable determines its size in memory and the range of values it can hold.

  • Integer Types: `int`, `char`, `short`, `long`. These can be modified with `signed` or `unsigned`.
  • Floating-Point Types: `float`, `double`, `long double`. Used for numbers with decimal points.
  • The void Type: Indicates that no value is available. Used for functions that do not return a value or for generic pointers.

4. Variables

Declaring and Initializing Variables

A variable is a name for a storage area. In C, all variables must be declared with a specific type before they are used.


// Variable declaration
int a, b, c;

// Initialization
a = 10;
char grade = 'A';
float score = 95.5f;

5. Constants

Defining Constants

Constants are fixed values that do not change. They can be defined using the `#define` preprocessor directive or the `const` keyword.


#include 

#define PI 3.14159
const int MAX_USERS = 100;

int main() {
   printf("Value of PI: %f\n", PI);
   printf("Max users: %d\n", MAX_USERS);
   return 0;
}

6. Operators

Common C Operators
  • Arithmetic: `+`, `-`, `*`, `/`, `%` (modulus)
  • Relational: `==`, `!=`, `>`, `<`, `>=`, `<=`
  • Logical: `&&` (AND), `||` (OR), `!` (NOT)
  • Bitwise: `&`, `|`, `^`, `~`, `<<`, `>>`

7. Decision Making

Conditional Logic

Control the flow of your program with conditional statements.

  • if-else: Execute different blocks of code based on a condition.
  • switch: Select one of many code blocks to be executed based on a variable's value.

8. Loops

Executing Code Repeatedly
  • for loop: Best used when the number of iterations is known.
  • while loop: Repeats as long as a condition is true.
  • do-while loop: Guarantees the loop body is executed at least once.

9. Functions

Modular Code with Functions

Functions are blocks of code that perform a specific task and can be reused throughout a program.

10. Arrays

Storing Data Collections

Arrays store a fixed-size sequence of elements of the same data type.


int numbers[5] = {10, 20, 30, 40, 50};

11. Pointers

Working with Memory Addresses

A pointer is a variable that stores the memory address of another variable, enabling direct memory access.


int var = 20;
int *ip = &var; // ip now holds the address of var

12. Strings

Strings as Character Arrays

In C, strings are represented as arrays of characters ending with a null character (`\0`).


char greeting[] = "Hello";

13. Structures

Complex Data Types

Structures (`struct`) allow you to group variables of different data types under a single name.


struct Book {
   char  title[50];
   char  author[50];
   int   book_id;
};
