C++ Tutorial

1. C++ Overview

What is C++?

C++ is a middle-level programming language developed by Bjarne Stroustrup at Bell Labs, starting in 1979. It was designed as an enhancement to the C language, adding object-oriented features. It is a compiled, statically-typed language known for its high performance.

Object-Oriented Programming (OOP)

C++ fully supports the four main pillars of object-oriented programming, which help in creating modular, reusable, and manageable code:

  • Encapsulation: Bundling data and the functions that operate on that data into a single unit (a class).
  • Data Hiding: Restricting access to certain members of an object to prevent unauthorized access.
  • Inheritance: Creating new classes (derived classes) from existing classes (base classes), enabling code reuse.
  • Polymorphism: The ability to present the same interface for differing underlying forms (data types).

2. Basic Syntax

First C++ Program

The entry point for any C++ application is the `main()` function. The `#include ` directive is used to include the standard input-output stream library.


#include 

// The using namespace std; line tells the compiler to use the standard namespace.
using namespace std;

// main() is where program execution begins.
int main() {
   cout << "Hello World"; // prints Hello World
   return 0;
}

3. Data Types

Primitive Built-in Types

C++ provides a rich set of built-in data types. The most common ones include:

  • bool: Stores either `true` or `false`.
  • char: Stores a single character.
  • int: Stores integer values.
  • float: Stores single-precision floating-point numbers.
  • double: Stores double-precision floating-point numbers.

4. Variable Types

Variable Definition and Declaration

A variable definition allocates storage for the variable, while a declaration only specifies its name and type. You can declare a variable multiple times, but it can only be defined once.


// Variable definition and initialization
int i = 10;
char c = 'A';

// Variable declaration (using extern)
extern int some_external_variable;

5. Operators

Common Operator Types
  • Arithmetic: `+`, `-`, `*`, `/`, `%` (modulus)
  • Relational: `==`, `!=`, `>`, `<`, `>=`, `<=`
  • Logical: `&&` (AND), `||` (OR), `!` (NOT)
  • Bitwise: `&` (AND), `|` (OR), `^` (XOR), `~` (NOT), `<<` (left shift), `>>` (right shift)

6. Loop Types

Types of Loops

Loops are used to execute a block of code repeatedly.

  • for loop: Executes a known number of times.
  • while loop: Executes as long as a condition remains true. Checks the condition before executing.
  • do-while loop: Similar to a while loop, but the code block is guaranteed to execute at least once before the condition is checked.

7. Decision Making

Conditional Statements
  • if statement: Executes code if a condition is true.
  • if-else statement: Executes one block of code if the condition is true, and another if it's false.
  • switch statement: Allows a variable to be tested for equality against a list of values (`case`).

8. Functions

Defining and Calling Functions

Functions are blocks of code that perform a specific task. They help organize code into manageable and reusable pieces.


#include 

// Function declaration (prototype)
int max(int num1, int num2);

int main() {
   int a = 100;
   int b = 200;
   int ret;
 
   // Calling the function
   ret = max(a, b);
   std::cout << "Max value is : " << ret << std::endl;
 
   return 0;
}

// Function definition
int max(int num1, int num2) {
   int result;
 
   if (num1 > num2)
      result = num1;
   else
      result = num2;
 
   return result; 
}

9. Pointers

What Are Pointers?

A pointer is a variable whose value is the memory address of another variable. They are essential for dynamic memory allocation and efficient data manipulation in C++.


int  var = 20;   // An actual variable
int  *ip;        // A pointer to an integer

ip = &var;      // Store the address of 'var' in the pointer

std::cout << "Value of var: " << var << std::endl;

// Access the value using the pointer
std::cout << "Value of *ip: " << *ip << std::endl;

10. Classes and Objects

Defining a Class

A class is a blueprint for creating objects. It bundles data members (attributes) and member functions (methods) into a single unit.


class Box {
   public:
      double length;   // Length of a box
      double breadth;  // Breadth of a box
      double height;   // Height of a box

      // Member function declaration
      double getVolume(void);
};

// Member function definition
double Box::getVolume(void) {
    return length * breadth * height;
}
