Java Tutorial

1. Java - Overview

What is Java?

Java is a high-level, object-oriented programming language developed by Sun Microsystems and released in 1995. It is designed to be "Write Once, Run Anywhere" (WORA), meaning compiled Java code can run on all platforms that support Java without recompilation.

Key Features
  • Object-Oriented: Everything in Java is an object.
  • Platform Independent: Compiled into platform-independent byte code that runs on a Java Virtual Machine (JVM).
  • Simple & Secure: Designed to be easy to learn and has built-in security features.
  • Robust & Multithreaded: Strong memory management and support for concurrent tasks.

2. Environment Setup

Local Environment Setup

To develop Java applications locally, you need to install the Java Development Kit (JDK). You can download it for free from Oracle or use an open-source alternative like OpenJDK. After installation, you must set the `JAVA_HOME` environment variable and add the JDK's `bin` directory to your system's `PATH`.

3. Basic Syntax

First Java Program

The entry point for any Java application is the `public static void main(String[] args)` method.


public class MyFirstJavaProgram {
    public static void main(String[] args) {
        System.out.println("Hello World"); // prints Hello World
    }
}
Syntax Rules
  • Case-Sensitive: `myVariable` and `MyVariable` are different.
  • Class Names: Start with an uppercase letter (e.g., `MyClass`).
  • Method Names: Start with a lowercase letter (e.g., `myMethod`).
  • File Name: Must exactly match the public class name (e.g., `MyClass.java`).

4. Objects & Classes

Understanding Objects and Classes

A **Class** is a blueprint for creating objects. It defines the properties (instance variables) and behaviors (methods) that its objects will have. An **Object** is an instance of a class.


public class Dog {
    // Instance variables
    String breed;
    int age;
    String color;

    // Methods
    void barking() {
        System.out.println("The dog is barking.");
    }
}

// Creating an object (instance) of the Dog class
Dog myDog = new Dog();
Constructors

A constructor is a special method that is used to initialize objects. It is called when an object is created using the `new` keyword. If you don't define a constructor, Java provides a default one.


public class Puppy {
    // Constructor with one parameter
    public Puppy(String name) {
        System.out.println("Passed Name is: " + name);
    }

    public static void main(String []args) {
        // The constructor is called here
        Puppy myPuppy = new Puppy("tommy");
    }
}

5. Basic Datatypes

Primitive vs. Reference Datatypes

Java has two categories of data types:

  • Primitive Types: The most basic data types available. They store the data directly. Examples: `int`, `byte`, `short`, `long`, `float`, `double`, `boolean`, `char`.
  • Reference Types: These variables store a reference (or memory address) to an object. Examples: `String`, Arrays, and any class you create.

6. Variable Types

Types of Variables
  • Local Variables: Declared inside a method. Their scope is limited to that method.
  • Instance Variables: Declared inside a class but outside any method. Each object of the class has its own copy of these variables.
  • Class/Static Variables: Declared with the `static` keyword. There is only one copy of this variable, shared among all objects of the class.

7. Modifier Types

Access & Non-Access Modifiers

Modifiers are keywords that you add to definitions to change their meanings.

  • Access Modifiers: Control the visibility. `public`, `protected`, `default`, `private`.
  • Non-Access Modifiers: Provide other functionality. `static`, `final`, `abstract`, `synchronized`, `volatile`.

8. Basic Operators

Common Operator Types
  • Arithmetic: `+`, `-`, `*`, `/`, `%` (modulus)
  • Relational: `==`, `!=`, `>`, `<`, `>=`, `<=`
  • Logical: `&&` (AND), `||` (OR), `!` (NOT)
  • Assignment: `=`, `+=`, `-=`, `*=`, `/=`

9. Loop Control

Types of Loops

Loops are used to execute a block of code repeatedly.

  • for loop: Executes a set number of times.
  • while loop: Executes as long as a condition is true.
  • do-while loop: Similar to a while loop, but the code block is executed at least once.
  • Enhanced for-each loop: A simpler way to iterate over arrays or collections.

10. Decision Making

Conditional Statements
  • if statement: Executes code if a condition is true.
  • if-else statement: Executes one block of code if the condition is true, and another if it's false.
  • if-else-if ladder: Used to decide among several alternatives.
  • switch statement: Allows a variable to be tested for equality against a list of values.
