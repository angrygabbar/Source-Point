MySQL Tutorial

1. What is MySQL?

The World's Most Popular Open Source Database

MySQL is an open-source relational database management system (RDBMS). It is a core component of the LAMP (Linux, Apache, MySQL, PHP) web application software stack. MySQL is known for its reliability, performance, and ease of use, making it a popular choice for web applications, from small projects to large-scale enterprise systems.

2. Common Data Types

MySQL Data Types

MySQL supports a wide range of SQL data types. Some of the most common include:

  • INT: A standard integer.
  • VARCHAR(size): A variable-length string.
  • TEXT: For long-form text strings.
  • DATE: Stores a date in YYYY-MM-DD format.
  • DATETIME: Stores a date and time in YYYY-MM-DD HH:MI:SS format.
  • TIMESTAMP: Automatically updated to the current timestamp when a row is modified.

3. Storage Engines

InnoDB vs. MyISAM

MySQL uses storage engines to handle the SQL operations for different table types. The two most common engines are:

  • InnoDB: The default storage engine. It is transaction-safe (ACID compliant) and supports foreign keys. Best for data-critical applications.
  • MyISAM: A simpler engine that does not support transactions or foreign keys. It is generally faster for read-heavy operations, such as in data warehousing.

4. Basic CRUD Operations

Create, Read, Update, Delete

These four basic functions of persistent storage are the bread and butter of database management.


-- CREATE (Insert a new record)
INSERT INTO users (name, email) VALUES ('John Doe', 'john.doe@example.com');

-- READ (Select records)
SELECT * FROM users WHERE name = 'John Doe';

-- UPDATE (Modify a record)
UPDATE users SET email = 'johndoe@newemail.com' WHERE name = 'John Doe';

-- DELETE (Remove a record)
DELETE FROM users WHERE name = 'John Doe';
