SQL Tutorial

1. SQL Overview

What is SQL?

SQL stands for Structured Query Language. It is the standard language for relational database management systems (RDBMS). SQL is used to perform operations such as creating databases, inserting data, fetching data, modifying data, and deleting data.

SQL Commands

SQL commands are broadly categorized into the following types:

  • DDL (Data Definition Language): Used to define the database structure. Examples: `CREATE`, `ALTER`, `DROP`.
  • DML (Data Manipulation Language): Used for managing data within schema objects. Examples: `SELECT`, `INSERT`, `UPDATE`, `DELETE`.
  • DCL (Data Control Language): Used to control access to data. Examples: `GRANT`, `REVOKE`.

2. Basic Syntax

Key Syntax Rules

SQL statements are built using a standard set of keywords and follow a specific structure. Here are some fundamental rules:

  • SQL is case-insensitive, meaning `SELECT` is the same as `select`. However, table and column names can be case-sensitive depending on the RDBMS.
  • Statements are terminated with a semicolon (;).
  • Keywords like `SELECT`, `FROM`, and `WHERE` are used to structure queries.

3. Data Types

Common Data Types

Each column in a database table requires a name and a data type. Common SQL data types include:

  • Numeric Types: `INT`, `BIGINT`, `DECIMAL(p, s)`, `FLOAT`.
  • Date and Time Types: `DATE`, `TIME`, `DATETIME`, `TIMESTAMP`.
  • Character String Types: `CHAR(n)`, `VARCHAR(n)`, `TEXT`.

4. Operators

Types of Operators

Operators are used in the `WHERE` clause to perform comparisons and logical operations.

  • Arithmetic Operators: `+`, `-`, `*`, `/`, `%`.
  • Comparison Operators: `=`, `!=`, `>`, `<`, `>=`, `<=`.
  • Logical Operators: `AND`, `OR`, `NOT`, `IN`, `BETWEEN`, `LIKE`.

5. CREATE Database

Creating a Database

This command is used to create a new database in the RDBMS.


CREATE DATABASE TestDB;

6. CREATE Table

Creating a Table

This command creates a new table within a database. You must define each column's name and data type.


CREATE TABLE Customers (
    ID   INT              NOT NULL,
    NAME VARCHAR (20)     NOT NULL,
    AGE  INT              NOT NULL,
    ADDRESS  CHAR (25),
    SALARY   DECIMAL (18, 2),       
    PRIMARY KEY (ID)
);

7. INSERT INTO Query

Adding New Records

The `INSERT INTO` statement is used to add new rows of data to a table.


INSERT INTO Customers (ID, NAME, AGE, ADDRESS, SALARY)
VALUES (1, 'Ramesh', 32, 'Ahmedabad', 2000.00);

8. SELECT Query

Retrieving Data

The `SELECT` statement is used to fetch data from a database. You can select specific columns or all columns using `*`.


-- Select specific columns
SELECT NAME, SALARY FROM Customers;

-- Select all columns
SELECT * FROM Customers;

9. WHERE Clause

Filtering Records

The `WHERE` clause is used to filter records and extract only those that fulfill a specified condition.


SELECT ID, NAME, SALARY
FROM Customers
WHERE SALARY > 2000;

10. UPDATE Query

Modifying Records

The `UPDATE` statement is used to modify existing records in a table.


UPDATE Customers
SET ADDRESS = 'Pune'
WHERE ID = 6;

11. DELETE Query

Deleting Records

The `DELETE` statement is used to delete existing records from a table.


DELETE FROM Customers
WHERE ID = 6;

12. Joins

Combining Tables

A `JOIN` clause is used to combine rows from two or more tables based on a related column between them.


SELECT C.ID, C.NAME, O.AMOUNT
FROM Customers AS C
INNER JOIN Orders AS O
ON C.ID = O.CUSTOMER_ID;

13. Unions

Combining Result Sets

The `UNION` operator is used to combine the result-set of two or more `SELECT` statements. It removes duplicate rows between the various `SELECT` statements.


SELECT City FROM Customers
UNION
SELECT City FROM Suppliers
ORDER BY City;
